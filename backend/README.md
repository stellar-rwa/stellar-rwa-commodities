# Backend Services

This directory contains off-chain backend services for the StellarCommodity platform.

## Services

- **Indexer**: Monitors the Stellar network for commodity token transactions.
- **Price Engine**: Aggregates prices from various oracles and external APIs.
- **KYC Processor**: Handles off-chain KYC validation logic.

## Getting Started

Each service is designed to run as a separate process or serverless function.
